# 🚀 Tekna.Rocks – QA Challenge | Joyce Cervantes

Este repositório contém a resolução do desafio técnico para a vaga de QA Analyst Jr. na [Tekna.Rocks](https://tekna.rocks), com foco em testes manuais, planejamento e sugestões de melhorias no produto.

## 📋 Sobre o desafio

O desafio propõe a análise de uma plataforma web de estimativa de economia com veículos elétricos. As tarefas envolvem:

- Desenvolvimento de casos de teste
- Identificação de pontos críticos e possíveis bugs
- Planejamento de implementação
- Sugestões de melhoria
- Estratégia de automação
